= OT Frame =
Byte 0: PARITY MSG-TYPE SPARE(0000)
Byte 1: DATAID
Byte 2: DATA VALUE MSB
Byte 3: DATA VALUE LSB

== Master MSG-TYPE ==
000 READ-DATA
001 WRITE-DATA
010 INVALID-DATA

== Slave MSG-TYPE ==
100 READ-ACK
101 WRITE-ACK
110 DATA-INVALID
111 UNKNOWN-DATAID
